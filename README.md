Load project into IDE of choice. For IntelliJ, select the 'EasyAnimator' program (visible to the left of the run button)

Select 'Edit Configurations'

Under Build and run, specify parameters in the input box on the right in the following format (ignore curly braces):

-if {fileName.txt} -iv interactive -speed {number 0-1000}

fileName.txt can be replaced with any of the animation .txt files available in the parent directory.
